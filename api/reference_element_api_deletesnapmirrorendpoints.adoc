= DeleteSnapMirror : points de terminaison
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
L'interface utilisateur web d'Element utilise `DeleteSnapMirrorEndpoints` pour supprimer un ou plusieurs noeuds finaux SnapMirror du système.



== Paramètre

Cette méthode dispose du paramètre d'entrée suivant :

|===
| Nom | Description | Type | Valeur par défaut | Obligatoire 


 a| 
SnapMirrorEndpointIDS
 a| 
Matrice d'ID de terminaux SnapMirror à supprimer
 a| 
tableau entier
 a| 
Aucune
 a| 
Oui

|===


== Valeurs de retour

Cette méthode n'a pas de valeurs de retour.
